# Star App

## Idea
Gesture-based social media app. Controlled with hand gestures.

## Main Screen
The main screen consists of 2 video screens.

### Right Side
Starts with 1 screen → then opens 3 screens.

### Left Side
Starts with 1 screen → then opens 3 screens.

## User Panel
Opens by swiping left-right with two fingers on the main 2-screen area.

## Future Features
- Like
- Save
- Gift
- Premium Share
- Messages

## Technologies
Will be built with Flutter
